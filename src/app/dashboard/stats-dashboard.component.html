Task T03_04 & T03_07 Explanation

This component demonstrates the use of a service to provide global data (T03_04). The stats you see are generated randomly by the DataService and injected into this component.

It also displays a list of heterogeneous entities (the stats), and the output is adjusted to the type of entity (T03_07).

This also demonstrates parent-child communication. Clicking on a stat card will select it and display the app-stat-details child component. The child component can emit an event back to the parent.
